PROJECT OVERVIEW

Project Rawwar revolutionizes the traditional fighting game genre by integrating typing mechanics as the core combat system. Players must type words correctly to deliver punches, creating an engaging and educational gaming experience that combines entertainment with skill development.

INNOVATIVE COMBAT MECHANICS

The game presents players with 5 random words that must be typed correctly. Each correct letter delivers a punch to the enemy, while mistakes result in taking damage. This creates a high-stakes typing environment that improves accuracy and speed.

EDUCATIONAL GAMING VALUE

Combines entertainment with learning, helping players improve their typing skills while enjoying engaging gameplay. The progressive difficulty system adapts to player skill level, ensuring continuous improvement and sustained engagement.

VISUAL FEEDBACK SYSTEM

Real-time visual and audio feedback provides immediate response to player actions. The combat animations and sound effects create an immersive experience that keeps players engaged and motivated to improve their performance.
