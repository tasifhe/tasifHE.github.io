PROJECT OVERVIEW

Drawnscape represents a breakthrough in physics-based puzzle gaming, combining innovative drawing mechanics with challenging level design. This project showcases advanced Unity2D development skills and creative problem-solving in mobile game design.

CORE GAMEPLAY MECHANICS

The game features two distinct modes: Draw Mode allows players to create intricate physics-based lines to guide the character, while Play Mode provides direct character control. This dual-mode system creates a unique gaming experience that challenges both creativity and strategy.

TECHNICAL INNOVATION

Built using Unity's advanced Physics2D system, the game implements real-time line drawing with accurate physics simulation. Custom algorithms ensure smooth performance across various Android devices while maintaining visual fidelity and responsive controls.

VISUAL DESIGN EXCELLENCE

The minimalist aesthetic focuses on clean geometry and vibrant colors, creating an engaging visual experience that doesn't overwhelm the core gameplay mechanics. Every visual element serves both aesthetic and functional purposes in the game experience.
